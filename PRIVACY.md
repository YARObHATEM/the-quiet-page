# Privacy

The Quiet Page is designed to work locally and offline.

- Journal entries, drafts, and settings are stored only in the app's user-data
  folder on the computer.
- The app contains no analytics, advertising, telemetry, accounts, or cloud
  synchronization.
- Fonts are bundled with the app, so opening the app does not contact a font
  provider.
- The app does not request camera, microphone, location, notification, or
  similar browser permissions.
- Data leaves the app only when the user explicitly exports a file or copies
  writing to the clipboard.

The project does not include or upload a user's local journal data.
