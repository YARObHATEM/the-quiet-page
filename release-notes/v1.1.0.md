# The Quiet Page v1.1.0

The first public release of The Quiet Page: a private, offline desktop writing
journal for Arabic and English.

## Downloads

- **Setup** installs the app normally and adds shortcuts.
- **Portable** runs without installation.

## Highlights

- Arabic RTL and English LTR writing
- Local library, search, editing, pinning, import, and export
- Focus mode, insights, themes, typography, and typewriter sounds
- Offline bundled fonts
- No accounts, analytics, advertisements, or network requests

## Windows notice

These community binaries are not code-signed, so Microsoft SmartScreen may
show an "unknown publisher" warning. Use `SHA256SUMS.txt` to verify the
downloaded file.
