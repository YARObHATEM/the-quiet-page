# The Quiet Page v1.1.2

The page now has a wider, fully offline type library for both English and
Arabic writing.

## Highlights

- Lora, Merriweather, JetBrains Mono, Nunito, and Playfair Display
- Amiri, Tajawal, Lateef, Cairo, and Scheherazade New for Arabic
- Separate English and Arabic font preferences
- Immediate font switching with automatic RTL/LTR application
- Local WOFF2 files only, with no runtime network access
