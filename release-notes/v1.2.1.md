# The Quiet Page 1.2.1

This release brings a cleaner, more spacious writing experience.

- **Tags removed** — the tags feature has been removed to keep the interface
  focused on what matters: your words.
- **Full-width reader** — entries now fill the available space when you open
  them in the library.
- **Cleaner library cards** — preview text removed, folder and metadata labels
  enlarged for easier scanning.
- **Sidebar polish** — the app title is now centered without the trailing dot.
- **Placeholder fix** — the "Write anything…" prompt now aligns perfectly with
  the cursor, no matter the font or line-height settings.
- **Forest audio balanced** — the Forest ambience now plays at the same volume
  as the other moods.
- **Library scrolling fixed** — the sidebar list and reader now scroll
  independently; reader buttons stay visible at the bottom of long entries.
